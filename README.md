# Daily Dev Tips: 오늘의 개발 Tip

### Install
```
git clone
cd daily-dev-tips
chmod 755 execute.sh
./execute.sh
```

### Environment
macOS만 지원

### Detail
터미널 접속 시 오늘의 개발 Tip을 출력한다.

